# Naming Conventions

## Product Names

- Product display name: `Sense Grain`
- Repository slug: `sense-grain`
- Package scope: `@sense-grain`

## Code Names

- Files and folders: kebab-case for app folders, camelCase for TypeScript modules when matching framework conventions.
- React components: PascalCase.
- Hooks: `useThingName`.
- Services: `thingService` or direct domain names such as `firebaseApp`.
- Firebase collections: camelCase plural names such as `grainReadings`, `storageBins`, and `deviceEvents`.

## Branch Names

- `feature/device-registration`
- `feature/grain-dashboard`
- `fix/firebase-rules`
- `docs/hardware-integration`
