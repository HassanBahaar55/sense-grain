# Sense Grain Web App

Next.js dashboard for operations teams, administrators, analytics, device management, and grain monitoring.

## Structure

```text
src/
  app/          Next.js app router
  components/   Shared dashboard components
  config/       Runtime configuration
  features/     Domain features
  lib/          Framework integrations
  services/     Firebase and hardware services
  styles/       Global styles and theme adapters
  types/        TypeScript types
```
